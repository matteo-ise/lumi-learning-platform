AI Strategy

AI Integration — No-RAG, Meta-Prompts, Vision

Why No RAG?

  • Curriculum context is small and stable — fits in system prompt
  • Eliminates vector DB infrastructure, latency, and cost
  • Gemini 2.0 Flash's 1M-token context window handles full lesson plans
  • Simpler architecture → faster iteration and fewer failure points

Meta-Prompt Construction

  • Wizard collects: name, grade (1–4), state (Bundesland), learning type
  • Backend builds a pupil profile string injected as {meta_prompt}
  • Every chat request includes: system rules + curriculum + pupil profile + subject
  • Enables personalised tone, vocabulary level, and example context

Vision Integration (Gemini Vision)

  • Student photos homework / worksheets → uploaded as multipart/form-data
  • Image passed directly to Gemini Vision via inline_data part
  • LUMI analyses tasks, identifies errors, then applies Socratic tutoring
  • Replaces OCR pipeline — no external vision model needed

Core Feature 1

Intelligent Tutoring — 3-Step Method + Socratic Principle

1

GRUNDLAGEN

Core concepts and vocabulary. Ensures shared understanding before progression.

→
2

VERTIEFUNG

Concrete examples and connections. Bridges theory to real understanding.

→
3

ÜBUNG

Child solves own task independently. LUMI prompts, never answers directly.

Socratic Principle

  • LUMI never reveals homework solutions — asks guiding questions instead
  • One step shown at a time — waits for the pupil before advancing
  • Hotkeys reduce typing burden: NEXT_STEP · SIMPLIFY · EXAMPLE
  • Exception: factual lists / vocabulary are given directly when requested

System Prompt Architecture

SYSTEM_PROMPT = RULES (pedagogical behaviour) + {lehrplan_kontext} (curriculum markdown) + {meta_prompt} (pupil profile) + {fach}/{thema} (subject/topic) + {lernziel} (learning goal) → Sent with every chat turn

Vision Pipeline

  • Child takes photo of worksheet on mobile/iPad
  • Frontend sends image as multipart/form-data to FastAPI
  • Backend encodes image and passes as inline_data to Gemini
  • Response includes task recognition + tutoring dialogue

Speech Pipeline

  • STT: browser captures speech → converted to text → sent to chat API
  • TTS: AI response text → speechSynthesis.speak() with child-friendly voice
  • Enables hands-free learning for dyslexic or early-stage readers
  • No additional API cost — fully browser-native

Engagement

Gamification & Engagement — LUMI Blast + Streaks

🚀

LUMI Blast — Math Shooter

  • HTML5 Canvas game embedded in the React app
  • Grade-specific arithmetic tasks (addition, subtraction, multiplication)
  • Player shoots the correct answer from multiple falling options
  • Difficulty scales with grade and performance
🔥

Streak System

  • Daily login and learning activity tracked
  • Visual streak counter motivates consistent return
  • Streak loss triggers gentle recovery prompts from LUMI

Engagement Design Principles

  • Immediate feedback: AI responds in <2s — no waiting for children
  • Hotkey shortcuts: Big tap targets replace typing ("Next Step", "Example")
  • Emoji anchors: Visual cues (📐🎯💡) reduce cognitive load
  • No grades, no pressure: Encouragement-only tone throughout
  • Avatar identity: LUMI dolphin companion builds emotional connection

Technical Implementation

  • Game loop: requestAnimationFrame at 60fps
  • Task generation: FastAPI endpoint returns grade-filtered problems
  • Score & streak: Firestore real-time updates
